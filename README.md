# Bank Loan Analytics Dashboard - Excel

## 📊 Project Overview
Built an interactive Excel dashboard to analyze 38.6k+ loan applications and evaluate loan performance, risk, and repayment behavior. The dashboard helps identify trends in funded amounts, defaults, and customer segments using slicers and dynamic KPIs.

## 🛠️ Tools & Skills
- **Excel**: Pivot Tables, Slicers, Conditional Formatting, Charts, Dashboard Design
- **Data Analysis**: Data Cleaning, KPI Calculation, Trend Analysis
- **Skills Demonstrated**: Financial Analysis, Risk Assessment, Data Visualization, Business Reporting

## 🎯 Key Features
- **Summary KPIs**: Total Applications, Funded Amount, Amount Received, Avg Interest Rate, Avg DTI with MTD & MoM comparison
- **Loan Status Breakdown**: Good vs Bad Loan issued with donut charts
- **Dynamic Filters**: Slicers for `Grade` and `Loan Purpose` for instant segmentation
- **Status Analysis**: Breakdown of Fully Paid, Charged Off, and Current loans across funded amount, interest rate, and DTI
- **Professional UI**: Dark theme layout optimized for presentations

## 📈 Key Insights from Dashboard
- **Total Applications**: 38.6k | **Total Funded Amount**: $435.8M | **Total Amount Received**: $473.1M
- **Good Loan Issued**: 86.18% ($370.2M funded)  
- **Bad Loan Issued**: 13.82% ($65.5M funded)
- **Avg Interest Rate**: 12.05% | **Avg DTI**: 13.33%
- **Top Performing Status**: Fully Paid loans account for $351.4M funded with 15.10% avg interest rate
- **Highest Risk**: Small Business and Renewable Energy loans show higher default rates

## 💡 How to Use
1. Download `financial_loan_data_excel.xlsx` from this repo
2. Open in Excel and go to the `SUMMARY DASHBOARD` sheet
3. Use slicers for `Grade` and `Purpose` to filter data instantly
4. All KPIs and charts update dynamically based on selection
